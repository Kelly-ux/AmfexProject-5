Monitor Function

  • Displays visual output, such as text, images, and videos.
  • Viewing web pages, documents, and multimedia content.
  • Interacting with graphical user interfaces (GUIs).
  • Playing games and editing images and videos.

Keyboard Function

  • Allows users to input text and commands via keys.
  • Typing documents, emails, and code.
  • Navigating through menus and selecting options.
  • Entering data into spreadsheets and databases.
  • Playing games and controlling multimedia applications.

Mouse Function

  • Allows users to interact with graphical user interfaces (GUIs).
  • A pointing device that allows users to control the cursor on the screen.
  • Clicking buttons, selecting options, and dragging objects.
  • Zooming in and out of images and documents.
  • Scrolling through web pages and documents.
  • Dragging and dropping objects.
